## Judul Program
**Program Kalkulator sederhana**

## Deskripsi
Program kalkulator ini meniru tampilan dan fungsionalitas kalkulator pada smartphone. Dibangun menggunakan PHP dengan antarmuka yang modern dan interaktif, serta dilengkapi dengan fitur riwayat perhitungan.

Program ini mendukung operasi matematika dasar (+, -, ×, ÷), persentase (%), dan desimal. Pengguna dapat berinteraksi melalui klik tombol atau keyboard.

## Fitur
- **Antarmuka Kalkulator HP**: Tampilan yang mirip dengan kalkulator smartphone modern
- **Operasi Dasar**: Penjumlahan, Pengurangan, Perkalian, Pembagian
- **Fitur Tambahan**: Persentase (%) dan Desimal
- **Input Keyboard**: Dukungan penuh untuk input dari keyboard
- **Riwayat Perhitungan**: Menyimpan 20 perhitungan terakhir dengan timestamp
- **Tombol Fungsi**: AC (Clear All), ⌫ (Backspace), dan % (Persentase)
- **Responsive Design**: Tampilan yang menyesuaikan di berbagai ukuran layar
- **Animasi Tombol**: Efek visual saat tombol ditekan
